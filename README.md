# TianTech WanQian AI 助手

WanQian 是由Tian-Tech独立运营的 AI 助手，致力于为用户提供高质量的智能对话和推理服务。我们的产品涵盖多种场景，满足不同用户的需求，从日常对话到复杂的推理任务，都能为您提供专业的支持。

## ⚠️ 重要声明
WanQian 所有服务都是调用 ChatGLM API 的。本产品名称为 WanQian，由开发者独立运营。WanQian 仅通过官方开放接口调用智谱 AI ChatGLM 大模型 API 提供 AI 能力，未对模型进行修改、训练及分发。WanQian 与智谱 AI 无官方关联、非官方产品、未获得官方代言或授权。

## ✨ 产品特性
WanQian 系列模型具有以下核心特性：
- **强大的推理能力**：基于先进的 ChatGLM 架构，能够处理复杂的推理任务
- **多轮对话支持**：保持上下文理解，提供连贯的对话体验
- **广泛的知识覆盖**：涵盖各个领域的知识，为用户提供准确的信息
- **快速响应**：优化的模型架构，确保快速的响应速度
- **思考过程可视化**：部分模型支持显示思考过程，帮助用户理解推理逻辑
